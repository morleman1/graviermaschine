\doxysubsection{Detailed Description}
The \doxylink{structL6474__BaseParameter__t}{L6474\+\_\+\+Base\+Parameter\+\_\+t} structure is used to set default initialization values for the library whenever a reset has been called or the library is used the first time. It can be filled with the global defaults by calling L6474\+\_\+\+Set\+Base\+Parameter and it then can be adapted or directly passed into L6474\+\_\+\+Initialize which always needs to be called after resetting the stepper driver or the library
